# Combination(조합)
### 서로 다른 n개중에서 순서 없이 k개를 뽑는 방법의 수

$C(n, k),\ nCk,\ ^nC_k$

$\binom{n}{k} = \frac{n!}{k!(n-k)!}$
-> n! (n개를 모두 나열하는 순서의 경우의 수), k!(그 중 선택한 k개의 순서가 겹치는 것 제거), (n-k)!(나머지 선택안한 n-k개 순서 겹치는 것 제거)


# Permutation(순열)
### 서로 다른 n개중에서 순서 있게 k개를 뽑는 방법의 수

$P(n, k),\ nPk,\ ^nP_k$

$P(n, k) = \frac{n!}{(n-k)!}$
